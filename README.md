# Fetal State Classification from CTG Data — Machine Learning (Python)

A machine learning pipeline to classify fetal state from **cardiotocography (CTG)** data, supporting clinical decision-making.

## What I did
- Built an **end-to-end pipeline** in Python: data preprocessing, exploratory analysis, and feature selection.
- Addressed **class imbalance** using SMOTE so the model wasn't biased toward the majority class.
- Trained and tuned **multiple classifiers** — Logistic Regression, Support Vector Machine, Random Forest, and XGBoost.
- Compared models using **accuracy, sensitivity, specificity, F1-score, and ROC-AUC** to choose the best performer.

## Tools
Python, Jupyter Notebook / Google Colab, pandas, scikit-learn, imbalanced-learn (SMOTE), XGBoost, Matplotlib / Seaborn
